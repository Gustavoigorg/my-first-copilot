# My First Copilot

Projeto desenvolvido durante o curso da Digital Innovation One (DIO) com objetivo de compreender como funcionam os modos de interação de um Copiloto baseado em IA.

## 🧠 Modos do Copiloto

O Copiloto pode operar em diferentes modos dependendo da necessidade do usuário.

### ❓ Ask
Modo utilizado para tirar dúvidas sobre código ou conceitos de programação sem alterar nenhum arquivo.

### ✏️ Edit
Modo utilizado para modificar ou melhorar código existente.

### 🧭 Plan
Modo usado para planejar soluções antes de implementar código.

### 🤖 Agent
Modo autônomo capaz de navegar pelo projeto e executar tarefas mais complexas.

### 📚 Study
Modo focado no aprendizado, guiando o usuário com explicações e perguntas reflexivas.
